# 首次初始化目录示例

```text
我的个人知识运营系统/
├─ 个人知识/
│  ├─ README.md
│  └─ INDEX.md
├─ 个人技能/
│  ├─ README.md
│  └─ INDEX.md
├─ 个人区域/
│  ├─ README.md
│  └─ INDEX.md
├─ 外部领域内容/
│  ├─ README.md
│  ├─ INDEX.md
│  └─ 2026-06-16.md
└─ 副业/
   ├─ README.md
   ├─ INDEX.md
   ├─ 已确认可尝试/
   │  ├─ README.md
   │  └─ INDEX.md
   ├─ 确认不做/
   │  ├─ README.md
   │  └─ INDEX.md
   └─ 建议的新项目/
      ├─ README.md
      └─ INDEX.md
```

## 说明

- `个人知识`：记录“我知道什么”
- `个人技能`：记录“我能做什么”
- `个人区域`：记录“我在哪个区域场景里应用这些能力”
- `外部领域内容`：记录“外部最近发生了什么”
- `副业`：记录“哪些项目值得试、哪些不做、哪些是新建议”
